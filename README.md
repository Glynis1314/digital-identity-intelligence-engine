# Digital Identity Intelligence Engine

A multi-platform OSINT and digital footprint intelligence platform built using Python, Selenium, Streamlit, and analytics dashboards.

## Features

- Multi-platform identity scanning
- GitHub / Reddit / LeetCode / HackerRank / Codeforces integration
- Social media scanning
- Confidence scoring
- Risk classification
- Analytics dashboard
- Screenshot intelligence gallery
- PDF exports
- Scan history persistence
- Stealth Selenium anti-detection

## Tech Stack

- Python
- Selenium
- Streamlit
- SQLite
- Plotly
- Pandas
